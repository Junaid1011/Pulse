Agent Role

Not a script.
An agent that reasons.

A script detects and alerts. PULSE autonomously decides which follow-up questions matter, runs them against Redshift, synthesises findings through Claude AI, and writes the fix — all without human instruction.

What makes it an agent
The 6-step pipeline is agentic because steps 3–5 are not predetermined. In step 3, PULSE decides which follow-up queries to run based on what step 1 returns. In step 4, Claude AI reads the diagnostic data and reasons about causation — not just pattern-matching. In step 5, it generates a human-ready fix message tailored to the specific publisher and issue. No human wrote that logic for each publisher. The agent did.
